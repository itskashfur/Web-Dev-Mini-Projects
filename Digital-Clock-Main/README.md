## 📝 Description
This project is a lightweight, responsive **Digital Clock** that displays the current time in a 12-hour format (HH:MM:SS AM/PM). It is part of the <a href="https://github.com/itskashfur/Web-Dev-Mini-Projects">Web-Dev-Mini-Projects</a> collection, focusing on DOM manipulation and CSS styling.

## 🚀 Features
<ul>
  <li><strong>Real-Time Updates:</strong> Synchronizes with your system clock every second.</li>
  <li><strong>AM/PM Toggle:</strong> Automatically detects and displays the time period.</li>
  <li><strong>Responsive UI:</strong> Works seamlessly on desktops, tablets, and smartphones.</li>
  <li><strong>Minimalist Design:</strong> Clean typography with a modern aesthetic.</li>
</ul>
