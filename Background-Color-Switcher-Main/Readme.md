# Background Color Switcher

A lightweight, interactive web application that allows users to dynamically change the background color of a webpage. This project is a part of the [Web-Dev-Mini-Projects](https://github.com/itskashfur/Web-Dev-Mini-Projects) repository, focusing on practical DOM manipulation and event handling.

## 🚀 Features
* **Instant Interaction:** Change the background color immediately upon clicking a color tile.
* **Modern UI:** A clean, centered layout with smooth hover effects.
* **Responsive Design:** Works seamlessly across different screen sizes.
* **Vanilla Stack:** Built entirely with HTML, CSS, and JavaScript—no external libraries required.

---

## 📂 Project Structure
The project is organized into three main files within the [Background-Color-Switcher-Main](https://github.com/itskashfur/Web-Dev-Mini-Projects/tree/main/Background-Color-Switcher-Main) directory:

| File | Description |
| :--- | :--- |
| `index.html` | The skeleton of the app, containing the color buttons and instructions. |
| `style.css` | The styling layer, handling the grid layout and button aesthetics. |
| `app.js` | The logic layer, using event listeners to update the body's background color. |
