## 🎮 Game Overview
This project is a part of the <a href="https://github.com/itskashfur/Web-Dev-Mini-Projects">Web-Dev-Mini-Projects</a> repository. It features a simple yet engaging gameplay experience where the player controls a paddle to bounce a ball and destroy a wall of bricks.

## 🚀 Features
* **Smooth Paddle Movement:** Controlled via mouse or keyboard.
* **Collision Detection:** Precise physics for ball-to-brick and ball-to-paddle interactions.
* **Score Tracking:** Real-time scoring as you break bricks.
* **Responsive Canvas:** Built using the HTML5 Canvas API for smooth rendering.

## 🛠️ Built With
* **HTML5:** Structure of the game and Canvas element.
* **CSS3:** Styling and layout.
* **JavaScript:** Game logic, animation frames, and collision physics.

## 📂 Project Structure
The folder contains the following essential files:
<ul>
  <li><code>index.html</code> - The main entry point and game container.</li>
  <li><code>style.css</code> - Visual styling for the game environment.</li>
  <li><code>ball.png</code> - Custom asset used for the game ball.</li>
</ul>

## 🕹️ How to Play
1.  Clone or download this repository.
2.  Open the <code>index.html</code> file in any modern web browser.
3.  Move the paddle to keep the ball in play.
4.  Break all the bricks to win!
