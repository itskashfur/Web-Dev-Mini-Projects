## 📝 About The Repository

This repository contains a variety of beginner-to-intermediate frontend web development projects. Each project is built from scratch without external libraries and focuses on fundamental concepts like DOM manipulation, event handling, responsive design, and CSS animations.

## 🚀 Getting Started

Follow these instructions to run any of the projects on your local machine.

<ol>
  <li>
    <strong>Clone the repository</strong>
    <pre><code>git clone https://github.com/itskashfur/Web-Dev-Mini-Projects.git</code></pre>
  </li>
  <li>
    <strong>Navigate into a specific project folder</strong>
    <pre><code>cd Web-Dev-Mini-Projects/BMI-calculator-main</code></pre>
  </li>
  <li>
    <strong>Run the application</strong>
    <br />
    Open the <code>index.html</code> file of the chosen project in your favorite web browser.
  </li>
</ol>
